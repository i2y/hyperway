gRPC-Web Example

Testing Hyperway's gRPC-Web support with a browser client

ℹ️ About this demo:
This page demonstrates gRPC-Web communication with a Hyperway server. The client uses the gRPC-Web protocol with base64 encoding (application/grpc-web-text) to communicate with the server directly from the browser.

Greeter Service

Endpoint: /grpcweb.example.v1.GreeterService/Greet

Calculator Service

Endpoint: /grpcweb.example.v1.GreeterService/Calculate

Protocol Details

This demo uses the gRPC-Web protocol with the following characteristics:

  • Content-Type: application/grpc-web-text (base64 encoded)
  • Message Format: 5-byte header (1 flag + 4 length) + protobuf payload
  • Trailers: Sent as a final message with flag 0x80
  • CORS: Enabled for browser access
